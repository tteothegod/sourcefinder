SOURCE FINDER

  • How it Works

    Source Finder works by sending html requests to google, parsing the html searching for a wikipedia page similar to that search. Then sending a http request to that wikipedia link and scalping that page for all its sources. All those sources are filtered searching for reputable databases, then the remaining sources are processed by a neural network i trained and deemed either credible or not

  • How to Use

    Enter a search term in the input box below and click "Search". The tool will find and classify Wikipedia sources based on credibility.
